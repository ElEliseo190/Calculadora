# Calculadora Estilo Iphone

Este programa Java implementa una calculadora interactiva inspirada en el diseño de iPhone. La calculadora puede realizar las operaciones básicas de matemáticas, incluyendo suma, resta, multiplicación y división. Además de las operaciones aritméticas estándar, la calculadora también tiene la capacidad de calcular el coseno y el seno de un número. La interfaz de usuario es intuitiva y fácil de usar, imitando la apariencia y la funcionalidad de una calculadora en un dispositivo iPhone.
